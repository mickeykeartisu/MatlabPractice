
	STFFile - STF-file related tools for MATLAB

This is a collection of tools to read/write STF files in MATLAB version of
STRAIGHT which Prof. Kawahara develops. These tools are useless for people
who do not use the MATLAB version. Current functions support not only
Legacy-STRAIGHT but also TANDEM-STRAIGHT.

readSTFFile: function to read an STF file.
writeSTFFile: function to write a STRAIGHT object to an STF file.
createSTFObject: function to create an STF object (for Legacy-STRAGIHT only).
testSTF: read/write test script used with Legacy-STRAIGHT functions.
testTandemSTF: read/write test script used with TANDEM-STRAIGHT functions.
